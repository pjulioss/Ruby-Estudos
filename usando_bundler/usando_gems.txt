Em projeto grandes não instalamos gem por gem, usamos o bundler para instalar
todas as gem (dependências) do projeto, veja o arquivo Gemfile para um exemplo
nele está a forma como declaramos todas as gems necessárias para nosso projeto

após a criação do nosso Gemfile dentro da pasta do projeto damos o comando (bundle)
esse comando ira olhar todas as gems listadas no Gemfile e fazer a instalação

* Para usar o bundle é necessário ter o (bundler) instalado que é o gerenciador
de gems.
